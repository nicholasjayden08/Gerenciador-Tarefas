# Task Manager

A simple Java application to manage your daily tasks efficiently.

## Features

- ✅ Add task
- ✅ Priority of the tasks
- ✅ Task category support
- ✅ Task difficulty support
- ✅ List all tasks
- ✅ List pending tasks
- ✅ List completed tasks
- ✅ Mark task as completed
- ✅ Delete task

## How to Run

Execute the class: `GerenciadorTarefasMain`

## Technologies Used

- Java
- ArrayList
- Object-Oriented Programming (OOP)

## Project Structure

- `Tarefa.java` - Task model
- `GerenciadorTarefas.java` - Task manager
- `GerenciadorTarefasMain.java` - Main menu
